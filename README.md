
# Offline Data Search Tool

**Author:** Gurkirat Singh

Offline Data Search Tool is a local, offline utility that allows searching across multiple data sources such as SQLite (.db) and CSV files using phone numbers or email addresses.

## Features
- Fully offline
- Modern full-screen UI
- Supports DB and CSV files
- Secure login screen

## Usage
python main.py

## Legal
Use only on data you own or have permission to analyze.
